Meteorite Landings Explorer

An interactive visualization of global meteorite landings using NASA open data. Features decade-based animation, class coloring, and mass scaling.

View Interactive Map View Code on GitHub

Key Findings

  • Reported meteorite landings increase sharply after the mid-20th century, reflecting improved detection and documentation rather than a true increase in events.
  • Ordinary chondrites (particularly L6) dominate modern records due to survivorship bias and advances in classification methods.
  • Grouping events by decade reveals global observation patterns more clearly than year-by-year animation.
